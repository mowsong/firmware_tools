# HEX/BIN 字节差异工具 — 用户指南

## 1）概述
`hex_diff_tool` 用于按字节比较两个文件：

- Intel HEX：`.hex`、`.ihex`、`.ihx`
- 二进制：`.bin`

可显示：
- 十六进制与 ASCII 视图
- 差异高亮
- 差异跳转（前后导航）
- 位变化统计（`0→1`、`1→0`）

---

## 2）启动方式

### 源码运行
```bash
python hex_diff_tool.py
```

### 打包程序
运行 `hex_diff_tool.exe`。

---

## 3）主界面说明

- **Open Left... / Open Right...**：加载左右文件
- **Only differences**：仅显示存在差异的行
- **Range [start, stop)**：只比较指定地址范围（十六进制）
- **差异导航按钮**：首个 / 上一个 / 下一个 / 最后一个
- **Refresh**：刷新比较结果
- **每侧 Reload / Close**：重载文件或清空该侧
- **右侧迷你图（mini-map）**：显示差异分布，可点击/拖动跳转
- **状态栏**：显示范围、行数、比较字节数、差异数、快捷键提示
- **位变化栏**：显示 `0→1`、`1→0` 与总位变化数量

---

## 4）加载文件

### 通过按钮
使用 **Open Left...** 和 **Open Right...**。

### 拖拽
- 拖到左侧视图：加载到 LEFT
- 拖到右侧视图：加载到 RIGHT
- 拖到窗口空白处：
  - 同时拖入 2 个及以上文件：第 1 个到 LEFT，第 2 个到 RIGHT
  - 只拖 1 个文件：优先填 LEFT，LEFT 已有文件则填 RIGHT

### BIN 偏移输入
加载 `.bin` 时会弹窗要求输入加载偏移（十进制或十六进制，如 `0x8000`）。
该偏移用于将文件字节索引映射到内存地址。

---

## 5）比较范围

范围格式为 `[start, stop)`：
- 包含 start
- 不包含 stop

支持输入示例：
- `8000`
- `0x8000`

规则：
- start/end 留空表示该侧无边界
- 必须满足 `stop > start`

---

## 6）差异视图解读

每行 16 字节：

`AAAAAAAA: xx xx ... xx  |ASCII........|`

- `AAAAAAAA`：行起始地址
- `xx`：字节值
- `..`：该地址无数据
- 红色高亮：差异字节
- 蓝色标记：当前定位的差异

ASCII 区域也会同步高亮差异位置。

---

## 7）导航

- 按钮：首个 / 上一个 / 下一个 / 最后一个差异
- 键盘：
  - `F7`：下一个差异
  - `Shift+F7`：上一个差异
  - `右/下方向键`、`PageDown`：下一个差异
  - `左/上方向键`、`PageUp`：上一个差异

---

## 8）仅差异模式

启用 **Only differences** 后：
- 无差异的行会隐藏
- 仅保留至少包含一个差异字节的行

---

## 9）位变化统计

显示格式：

`Bit changes: 0→1: N bits    1→0: M bits    Total: T`

统计范围：
- 在当前比较范围内
- 左右两侧该地址都有数据
- 且字节值不同

---

## 10）重载与关闭

- **Reload**：从磁盘重新读取同一路径文件
- **Close (✕)**：清空该侧文件，另一侧保留显示

适合反复构建固件后快速复查差异。

---

## 11）常见错误

- **不支持的文件类型**：请使用 `.hex/.ihex/.ihx/.bin`
- **BIN 偏移无效**：请输入十进制或十六进制（`0x...`）
- **范围输入无效**：请输入十六进制，并确保 `stop > start`
- **加载失败**：文件不存在、权限问题或 HEX 格式错误

---

## 12）使用建议

- 批量拖拽可加快对比流程。
- 使用范围过滤聚焦 Bootloader / App 区段。
- 大文件建议开启 **Only differences**。
- 通过右侧迷你图快速跳转到差异密集区域。
